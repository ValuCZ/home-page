Splunk - Apache


An implementation showcase by Ondřej Valů

Phase 1: Setting Up Apache

Configured Apache server to host multiple virtual hosts, including obtaining SSL certificates and setting up proxy pass to a Python Flask server.

Phase 2: Load Balancing and Health Checks

Implemented a round robin load balancer with health checks to ensure high availability and reliability of the service.

Phase 3: Logging and Monitoring

Set up a centralized logging system using Splunk to collect and analyze logs from the application, ensuring effective monitoring and troubleshooting.

Phase 4: Final Showcase

Compiled the entire project into a modern showcase webpage, highlighting the skills and technologies used in each phase.
